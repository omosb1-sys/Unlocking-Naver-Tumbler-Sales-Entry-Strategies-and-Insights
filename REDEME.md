# 🛍️ 네이버 쇼핑 텀블러 데이터 분석 프로젝트

이 프로젝트는 **네이버 쇼핑에서 '텀블러' 관련 상품 데이터**를 수집하고,  
가격, 용량, 브랜드별 특징을 분석한 결과를 시각화한 주피터 노트북입니다.

---

## 📂 파일 구성

| 파일명 | 설명 |
|--------|------|
| `텀블러분석(네이버쇼핑).ipynb` | 메인 분석 노트북 (데이터 수집, 전처리, 시각화 포함) |
| `텀블러분석(네이버쇼핑) 2.ipynb` | 보조 분석 버전 |
| `tumbler_analyzed.csv` | 정제된 네이버 쇼핑 텀블러 데이터 |
| `capacity_price.png` | 용량별 평균 가격 시각화 결과 |
| `price_distribution.png` | 전체 가격 분포 그래프 |

---

## 🧠 분석 개요

1. 네이버 쇼핑에서 “텀블러” 검색 결과 크롤링  
2. 상품명, 가격, 용량, 브랜드 정보를 추출  
3. 데이터 전처리 및 이상치 제거  
4. 용량과 가격의 관계, 브랜드별 가격대를 시각화

## 💡 주요 인사이트
- **대용량 제품일수록 평균 단가가 높음**
- **특정 브랜드(예: 스타벅스, 써모스 등)** 이 프리미엄 시장 주도
- **중저가 브랜드는 400~500ml 구간 제품에 집중**
